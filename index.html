Every robot walking today was hand-coded to walk. Every one. The gait cycles, the balance reflexes, the reward functions — all written by engineers, for one body, one task, one terrain.

Change the body, start over. Change the terrain, retrain. Change the task, redesign. Billions poured into controllers that shatter the moment something changes.

That is not intelligence. That is choreography.

A newborn does not know it has legs.

It flails. It kicks. It feels the ground, falls, feels it again. Weeks pass. It crawls. Months pass. It walks. No one programmed a gait cycle. No one designed a reward function. No one told it how many limbs it has. It discovered all of that — by building a model of its own body from raw sensation.

Put that toddler on the moon and it would relearn in minutes. Not because it memorized Earth. Because it knows how to explore a body in a world.

This is what we build.

20n: Self-Supervised Body Discovery for Universal Robot Intelligence

We are 20n. We build a brain that discovers how to control any body it is given.

Not another framework. Not another startup wrapping a language model around a robotic arm. A fundamentally different approach: a single architecture that takes sensory input from any body — humanoid, quadruped, hexapod — builds its own model of the physics, and learns to move. From scratch. Every time.
